## 微信小程序开发入门教程

> 无需编程基础，手把手教你发布第一个小程序

### 一、开发前准备

#### 1. 注册账号

- 访问[微信公众平台](https://mp.weixin.qq.com/)→ 立即注册 → 选择「小程序」类型
- 完成主体认证（个人/企业均可）

#### 2. 安装开发工具

- 下载[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（稳定版）
- 安装后扫码登录（绑定小程序账号）

#### 3. 创建项目

- 打开开发者工具 → 新建项目
- 填写： AppID（在公众平台「设置-开发设置」中获取） 项目名称（如：`MyFirstMiniProgram`） 项目目录（本地空文件夹） 勾选「不使用云服务」（后续可手动添加）

### 二、小程序核心概念解析

#### 1. 文件结构

```
├── pages/               # 页面目录
│   ├── index/            # 首页
│   │   ├── index.js      # 逻辑层
│   │   ├── index.json    # 配置文件
│   │   ├── index.wxml    # 视图层（类似HTML）
│   │   └── index.wxss    # 样式表（类似CSS）
│   └── logs/             # 日志页
├── utils/                # 工具函数
├── app.js                # 全局逻辑
├── app.json              # 全局配置（页面路径、窗口样式等）
├── app.wxss              # 全局样式
└── project.config.json   # 项目配置
```

#### 2. 双线程架构

| 线程       | 运行环境       | 职责                    |
| ---------- | -------------- | ----------------------- |
| **逻辑层** | JavaScript引擎 | 处理业务逻辑、数据请求  |
| **视图层** | WebView渲染    | 渲染页面UI（WXML+WXSS） |

> 两线程通过**数据绑定**和**事件系统**通信

#### 3. 数据驱动视图

```
// index.js
Page({
  data: { count: 0 }, // 初始化数据
  onIncrement() {
    this.setData({ count: this.data.count + 1 }) // 更新数据
  }
})
<!-- index.wxml -->
<view>当前计数：{{count}}</view>
<button bindtap="onIncrement">+1</button>
```

### 三、开发实战：TodoList小程序

#### 步骤1：配置全局文件

编辑 `app.json`定义页面和窗口样式：

```
{
  "pages": ["pages/index/index"],
  "window": {
    "navigationBarTitleText": "TodoList",
    "navigationBarBackgroundColor": "#007AFF"
  }
}
```

#### 步骤2：编写首页

**index.wxml**（视图层）：

```
<view class="container">
  <input value="{{inputValue}}" bindinput="onInput" placeholder="输入任务"/>
  <button bindtap="addTask">添加</button>
  
  <view wx:for="{{tasks}}" wx:key="id">
    <checkbox checked="{{item.done}}" bindchange="toggleTask" data-id="{{item.id}}"/>
    <text class="{{item.done ? 'done' : ''}}">{{item.name}}</text>
    <button size="mini" bindtap="deleteTask" data-id="{{item.id}}">删除</button>
  </view>
</view>
```

**index.wxss**（样式层）：

```
.container { padding: 20rpx; }
.done { text-decoration: line-through; color: #999; }
```

**index.js**（逻辑层）：

```
Page({
  data: {
    inputValue: '',
    tasks: [] // 任务数组 [{id: 1, name: '学习', done: false}]
  },

  // 输入框变化
  onInput(e) {
    this.setData({ inputValue: e.detail.value });
  },

  // 添加任务
  addTask() {
    if (!this.data.inputValue.trim()) return;
    const newTask = {
      id: Date.now(),
      name: this.data.inputValue,
      done: false
    };
    this.setData({
      tasks: [...this.data.tasks, newTask],
      inputValue: ''
    });
  },

  // 切换任务状态
  toggleTask(e) {
    const id = e.currentTarget.dataset.id;
    this.setData({
      tasks: this.data.tasks.map(task => 
        task.id === id ? {...task, done: !task.done} : task
      )
    });
  },

  // 删除任务
  deleteTask(e) {
    const id = e.currentTarget.dataset.id;
    this.setData({
      tasks: this.data.tasks.filter(task => task.id !== id)
    });
  }
});
```

```

  method: 'GET',

  fail(err) {
    console.error('请求失败:', err);
  }

```

  }

```

```

db.collection('todos').add({
  data: { name: '云开发任务', done: false }
});

### 常见问题解答

**Q1：小程序和网页开发有什么区别？**

- 小程序使用**WXML/WXSS**（类HTML/CSS），但语法有差异（如`<view>`代替`<div>`）
- 不支持DOM/BOM操作，通过**数据绑定**更新视图
- 需遵循微信的**安全规范**（如域名白名单）

**Q2：如何学习更高效？**

- 模仿优秀开源项目（GitHub搜索`miniprogram-demo`）
- 从修改现成模板开始（微信开发者工具提供模板市场）
- 关注「微信开放社区」每日更新的技术文章
