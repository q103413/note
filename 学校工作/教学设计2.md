## 《JavaScript前端开发》课程教学设计方案

### 一、 课程基本信息

* **课程名称：** JavaScript 前端开发（或 JavaScript 与 jQuery 实战）


* **适用专业：** 计算机科学与技术、软件工程、Web 前端开发、数字媒体技术。
* **课程性质：** 专业核心课 / 必修课。

### 二、 教学指导思想

本课程坚持**“任务驱动、案例教学、德技并修”**的理念。以实际网页交互效果为载体，将枯燥的语法知识融入到京东秒杀、购物车等真实业务场景中。教学过程强调“手脑并用”，通过 Chrome 浏览器的调试实践，培养学生的逻辑分析能力和工程实践素养。

### 三、 教学目标

* **知识目标：** 掌握 JS 变量、流程控制、函数；深入理解 Date、Math、Array、JSON 等对象；掌握 DOM/BOM 编程及 ES6 新特性；熟练使用 jQuery 库。


* **能力目标：** 能够独立实现京东秒杀倒计时、表单验证、动态购物车交互、响应式瀑布流等特效。


* **素质目标：** 培养规范的代码编写习惯、严谨的逻辑调试思维以及自主解决复杂技术问题的能力。

### 四、 内容结构

课程内容按照由浅入深、由原生到框架的逻辑分为五大核心板块：

| 板块         | 对应章节  | 核心知识点                                   |
| ------------ | --------- | -------------------------------------------- |
| **基础语法** | 第 1-2 章 | 环境搭建、调试工具、变量、流程控制、函数定义 |
| **对象编程** | 第 3 章 | 内置对象（Date/Math/Array/String）、JSON、正则表达式|
| **交互模型** | 第 4-5 章 | BOM 编程、本地存储、DOM 节点操作、网页偏移与滚动属性|
| **现代 JS** | 第 6 章 | ES6 箭头函数、类（Class）、数组与解构赋值|
| **jQuery 实战** | 第 7-11 章 | 选择器、DOM 操作、动画效果、插件应用（瀑布流等）|

### 五、 教学安排与实战任务

本课程通过一系列递进式的任务开展教学活动：

1. **入门阶段：** 学习在页面引入 JS，利用 Chrome 调试工具显示个人信息。


2. **逻辑训练：** 通过“学生考评系统”掌握 Switch 语句，通过“猜数游戏”掌握循环结构。


3. **对象应用：**

* 利用 `Date` 对象实现**京东秒杀效果**。


* 利用 `JSON` 对象实现**疫情防控新闻展示**。


* 利用正则实现**注册页面信息验证**。




4. **动效开发：**

* 使用 DOM 制作**复选框全选**与**动态表格**。


* 实现**京东商品切换选项卡**与**手风琴动画**。




5. **jQuery 高级：**

* 仿制**京东购物车前端局部更新**与前后端交互效果。


* 应用插件实现**响应式网格瀑布布局**。





### 六、 教学资源

* **开发环境：** VS Code 或 HBuilderX、Chrome 浏览器（含开发者工具）。


* **教学素材：** 课程配套的京东商城页面模版、JSON 数据接口、jQuery 官方库及插件。


* **参考文档：** MDN Web Docs、jQuery API 文档。

### 七、 教学策略与方法

* **任务驱动法：** 每个知识点对应一个具体的“任务单元”，如通过制作“简易计算器”来讲解函数和 ES6 箭头函数。


* **案例演示法：** 教师实时演示京东购物车等商业案例的逻辑拆解与代码编写过程。


* **调试驱动教学：** 将 Chrome 调试器的使用贯穿始终，教授学生如何查看控制台报错、设置断点。



### 八、 课程评价

* **过程性考核 (60%)：** 包括课堂实训任务的完成情况（如“随机选号”、“电梯导航”等任务的运行效果）。


* **期末综合考核 (40%)：** 包含 JavaScript 基础知识复习及一个综合性网页交互项目的开发。
