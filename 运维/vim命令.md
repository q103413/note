# 📝 Vim 常用命令速查

## 🔄 模式切换

- **普通模式**（默认模式，用来移动、复制、删除等操作）
- **插入模式**（输入文本）：
  - `i` → 在光标前插入
  - `a` → 在光标后插入
  - `o` → 在下一行新开一行
- **命令模式**（输入 `:` 开始，用于保存/退出等）
- **可视模式**（选择文本）：`v`

------

## 📂 文件操作

- `:w` → 保存文件
- `:q` → 退出 Vim
- `:q!` → 强制退出，不保存
- `:wq` 或 `:x` → 保存并退出
- `:e 文件名` → 打开文件
- `:saveas 新文件名` → 另存为

------

## 📍 光标移动

- `h` → 左移
- `l` → 右移
- `0` → 行首
- `^` → 行首第一个非空字符
- `$` → 行尾
- `gg` → 文件开头
- `G` → 文件末尾
- `:n` → 跳转到第 n 行
- `Ctrl + f` → 向下翻页
- `Ctrl + b` → 向上翻页

------

## ✂️ 编辑操作

- `x` → 删除光标所在字符
- `dd` → 删除当前行
- `ndd` → 删除 n 行
- `yy` → 复制当前行
- `nyy` → 复制 n 行
- `p` → 粘贴到光标后
- `u` → 撤销
- `Ctrl + r` → 重做

------

## 🔍 搜索与替换

- `/关键字` → 向下搜索
- `?关键字` → 向上搜索
- `n` → 下一个结果
- `N` → 上一个结果
- `:%s/旧/新/g` → 全文替换
- `:n,m s/旧/新/g` → 第 n 到 m 行替换

------

## 📑 多文件与窗口

- `:split 文件名` → 水平分屏
- `:vsplit 文件名` → 垂直分屏
- `Ctrl + w` + `方向键` → 在分屏之间切换
- `:tabnew 文件名` → 新建标签页
- `gt` → 切换到下一个标签页

------

## 🏆 实用技巧

- `.` → 重复上一次操作
- `>>` → 缩进当前行
- `<<` → 取消缩进
- `:%y+` → 复制全文到系统剪贴板（需支持 `+clipboard`）
- `:set number` → 显示行号
- `:set nonumber` → 隐藏行号
