------

## 教学绝活流程（6 分钟）

### 第 1 分钟：展示静态页面

> 这是一个企业中最常见的商品展示页面，目前是**静态的**。

------

### 第 2–4 分钟：现场编码（核心）

现场手敲（示例逻辑）：

```html
<button id="btn">点击切换</button>
<div id="box">商品详情</div>
const btn = document.getElementById("btn");
const box = document.getElementById("box");

btn.onclick = function () {
  box.classList.toggle("active");
};
.active {
  color: red;
}
```

**强调：**

- 原生 JavaScript
- 无框架
- 逻辑清晰
- 学生看得懂、学得会

------
