### 常用快捷键（Windows/macOS）

------

#### 编辑相关

- **复制行**：`Shift + Alt + ↑` 或 `Shift + Alt + ↓`（Windows），`Shift + Option + ↑` 或 `Shift + Option + ↓`（macOS）
  - 快速复制光标所在的一整行。
- **删除行**：`Ctrl + Shift + K`（Windows/macOS）
- **移动行**：`Alt + ↑` 或 `Alt + ↓`（Windows），`Option + ↑` 或 `Option + ↓`（macOS）
  - 快速将光标所在行上下移动。
- **多行选择**：`Alt + Click`（Windows），`Option + Click`（macOS）
  - 按住 `Alt`（或 `Option`）并点击多个位置，可以创建多个光标，同时进行编辑。
- **查找替换**：`Ctrl + F`（Windows），`Cmd + F`（macOS）
- **在文件中查找替换**：`Ctrl + H`（Windows），`Cmd + H`（macOS）
- **格式化代码**：`Shift + Alt + F`（Windows），`Shift + Option + F`（macOS）
  - 自动格式化整个文件，让代码保持整洁。

------



#### 导航与搜索



- **打开文件**：`Ctrl + P`（Windows），`Cmd + P`（macOS）
  - 快速搜索并打开项目中的任何文件。
- **跳转到行**：`Ctrl + G`（Windows），`Cmd + G`（macOS）
  - 输入行号即可快速跳转。
- **跳转到定义**：`F12`
  - 快速跳转到函数或变量的定义位置。
- **切换文件**：`Ctrl + Tab`（Windows），`Ctrl + Tab`（macOS）
- **侧边栏开关**：`Ctrl + B`（Windows），`Cmd + B`（macOS）
- **打开终端**：`Ctrl + `` `（Windows），`Ctrl + `` `（macOS）
  - 快速打开或关闭集成终端。

------



#### 多光标和选择



- **添加光标到下一行**：`Ctrl + Alt + ↓`（Windows），`Cmd + Option + ↓`（macOS）
- **添加光标到上一行**：`Ctrl + Alt + ↑`（Windows），`Cmd + Option + ↑`（macOS）
- **选中相同内容**：`Ctrl + D`（Windows），`Cmd + D`（macOS）
  - 选中一个词，连续按 `Ctrl + D`，可以逐个选中下一个相同的词，方便批量修改。

希望这些快捷键能帮助你更高效地使用 VS Code！
